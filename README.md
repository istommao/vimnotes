## 到最后一行

```shell

```

```shell

```

```shell
A
```

    k

```shell

# 向下滚动一行
Ctrl + e

Ctrl + y

Ctrl + d


# 向上滚动半屏
Ctrl + u


# 向下滚动一屏
Ctrl + f


# 向上滚动一屏
Ctrl + b
```
