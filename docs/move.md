# 移动

## 上下左右 移动

```shell
    ^
    k
< h    l >
    j
    V
```

## 移动到本行第一个字符 0（数字 0）

```shell
0
```

## 移动到本行第一个非空白字符上 ^

```shell
^
```

## 移动到行尾 \$

```shell
$

# 移动到下面5行的行尾
5$
```

## 到第一行

```shell
gg
```

## 到最后一行

```shell
G
```
